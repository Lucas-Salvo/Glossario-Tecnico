Glossário Técnico: HTML, CSS e JS

Este glossário tem como objetivo apresentar os principais termos relacionados a HTML (HyperText Markup Language), CSS (Cascading Style Sheets) e JS (Java Script). 3 dos conhecimentos mais básicos para se desenvolver uma pagina web. Ao compreender esses conceitos, o leitor poderá desenvolver uma base sólida para explorar o desenvolvimento web, interpretando melhor códigos, tutoriais e documentos técnicos. Este material é voltado tanto para iniciantes quanto para aqueles que desejam revisar os conceitos mais importantes dessas linguagens.

HTML (HyperText Markup Language)

Linguagem de marcação usada para estruturar o conteúdo de páginas da web.
CSS (Cascading Style Sheets)

Linguagem usada para definir o estilo visual de páginas HTML, como cores, fontes e layout.
JS (Java Script)

linguagem de programação usada principalmente para deixar os sites interativos e dinâmicos.

Conceitos HTML

Footer

Rodapé da página ou seção.

Form

Formulário para envio de dados do usuário.

Input

Campo de entrada de dados (texto, senha, checkbox, etc).

Table

Tabela de dados.

Vídeo

Reprodutor de vídeo.

Áudio

Reprodutor de áudio.

Script

Código JavaScript embutido ou linkado.

Iframe

Janela para carregar outra página dentro da atual.

I

Texto em itálico ou usado para ícones (ex: Font Awesome).

Main

Conteúdo principal único da página.

Conceitos css

Color

Define a cor do texto.

Background-Color

Define a cor de fundo de um elemento.

Font-Size

Define o tamanho da fonte do texto.

Font-Family

Define a família de fontes usada no texto.

Text-Align

Alinha o texto (esquerda, direita, centro, justificado).

Margin

Define o espaço externo ao redor do elemento.

padding

Define o espaço interno entre o conteúdo e a borda do elemento.

Border

Define a borda do elemento (largura, estilo, cor).

Display

Controla como o elemento é exibido (inline, block, flex, etc.).

widht & Height

Define a largura e altura do elemento.
